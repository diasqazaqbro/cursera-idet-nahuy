# Удаление слова из текста

Веб-приложение для удаления определенного слова из текста.

## Установка

```bash
npm install
```

## Настройка переменных окружения

Создайте файл `.env` в корне проекта на основе `.env.example`:

```bash
cp .env.example .env
```

Заполните переменные в `.env`:
- `NEXT_PUBLIC_OPENAI_API_KEY` - ваш API ключ от OpenAI
- `NEXT_PUBLIC_CORRECT_PASSWORD` - пароль для доступа к AI функционалу

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Использование

1. Введите слово, которое нужно удалить
2. Введите ваш текст
3. Результат появится автоматически
4. Используйте кнопку "Копировать" для копирования результата

## Технологии

- React
- Vite
- Tailwind CSS
